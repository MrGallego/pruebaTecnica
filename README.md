Proyecto Ascensor
Descripción

-Este proyecto simula y controla un ascensor mediante un panel web.
-El frontend está hecho en AngularJS 1.x.
-El backend es un API en .NET que maneja el estado del ascensor y las solicitudes de pisos.
-Permite visualizar el piso actual, la dirección del movimiento, estado de las puertas y solicitudes pendientes.
-Incluye botones de control manual: abrir/cerrar puertas, iniciar/detener ascensor y solicitar piso.

----------

Funcionalidades:
Consultar el estado del ascensor en tiempo real.
Mostrar lista de solicitudes pendientes.
Solicitar un piso mediante input y botón.
Control manual del ascensor: abrir/cerrar puertas, iniciar/detener.
Actualización automática del panel cada segundo.


Requisitos:
Navegador moderno (Chrome, Edge, Firefox).
Backend .NET corriendo en https://localhost:7023.
AngularJS se carga desde CDN, no requiere instalación adicional.


Use estos Script para crear las tablas en su bd local:

CREATE TABLE solicitudes (
    id NUMBER GENERATED BY DEFAULT ON NULL AS IDENTITY PRIMARY KEY,
    piso NUMBER NOT NULL,
    atendida NUMBER(1) DEFAULT 0,  -- 0 = pendiente, 1 = atendida
    fecha_solicitud DATE DEFAULT SYSDATE
);

CREATE TABLE log_operaciones (
    id NUMBER GENERATED BY DEFAULT ON NULL AS IDENTITY PRIMARY KEY,
    descripcion VARCHAR2(255),
    fecha DATE DEFAULT SYSDATE
);
